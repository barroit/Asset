== Usage

run `./brtool` to generate the corresponding file symlinks

create a rclone config named 'onedrive', with type 'onedrive', then run `./ntfs`

run `sudo dpkg-reconfigure keyboard-configuration` to set a general 105 keys
keyboard with language set to japanese, and run `xmodmap .Xmodmap`

run `setup-environment` to get a comfortable environment
